# commands
- `nmap <leader>s` get keymap
- `set option?` get option value
- `set filetype?` get detected filetype
- `<n> ^w w` go to nth window
- `^w p` go to last accessed window
- `{n}^g` get cur path, n > 1 to see buffer name
- `InspectTree` open syntax tree
- `vim.diagnostic.hide()`
- `^e` cancel dropdown
